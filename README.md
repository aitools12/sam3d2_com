# Sam 3D 2.0

Sam 3D 2.0 is an AI 3D model generator for text-to-3D and image-to-3D workflows. It helps users turn prompts and reference images into previewable, export-ready 3D assets for product visuals, characters, scenes, and creative production.

## What is Sam 3D 2.0

Sam 3D 2.0 is a browser-based AI 3D generation platform for people who need usable 3D output without starting every object from zero. The public product positioning is centered on prompt-based 3D generation, image-to-3D creation from reference inputs, browser preview, and flexible export workflows.

## Why use Sam 3D 2.0

Sam 3D 2.0 is useful when a team needs faster 3D iteration instead of a slower manual-first pipeline. Designers can test multiple directions from one prompt. Marketers can produce 3D-style visuals for product launches. Creators can move from a still reference to a usable 3D concept more quickly.

It is especially relevant for:

## How Sam 3D 2.0 works

1. Enter a text prompt or upload a source image.
2. Generate a 3D draft and preview it in the browser.
3. Refine the input and export the strongest result for downstream work.
